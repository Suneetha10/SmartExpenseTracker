# SmartExpenseTracker

![SmartExpenseTracker Banner](https://via.placeholder.com/1200x300?text=SmartExpenseTracker+Banner)

## Overview
SmartExpenseTracker is a user-friendly personal finance application designed to help you effortlessly manage and visualize your expenses. With insightful charts and intuitive categorization, you can take control of your spending and make better financial decisions.

## Features
- Add, edit, and delete expense entries
- Categorize expenses with customizable tags
- Visualize spending with interactive charts and graphs
- Monthly and yearly summary reports
- Responsive design for web and mobile

## Tech Stack
- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB
- Charts: Chart.js

## Installation
1. Clone the repository:

   ```bash
   git clone https://github.com/Suneetha10/SmartExpenseTracker.git
   ```

2. Navigate into the project directory:

   ```bash
   cd SmartExpenseTracker
   ```

3. Install dependencies for backend and frontend:

   ```bash
   cd backend
   npm install
   cd ../frontend
   npm install
   ```

4. Setup environment variables in `backend/.env` (e.g., MongoDB URI, PORT).

5. Run backend server:

   ```bash
   cd ../backend
   npm start
   ```

6. Run frontend app:

   ```bash
   cd ../frontend
   npm start
   ```

## Usage
- Open the app in your browser at http://localhost:3000
- Register or log in to start tracking your expenses
- Use the dashboard to add expenses and view reports
